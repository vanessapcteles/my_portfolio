## 🚀 Live Demo

[**View Live Portfolio**](https://my-portfolio-theta-three-m4ngmtp639.vercel.app)

## 🛠️ Technologies Used

This project was built from scratch using semantic HTML, modern CSS, and vanilla JavaScript.

*   **HTML5**: Semantic structure and layout.
*   **CSS3**:
    *   Flexbox & CSS Grid for responsive layout.
    *   CSS Variables (`:root`) for consistent theming (Dark Mode).
    *   Keyframe animations (floating effects) and transitions.
    *   Mobile-first approach.
*   **JavaScript (ES6+)**:
    *   DOM Manipulation.
    *   Dynamic page title (Page Visibility API).
    *   "Back to Top" scroll functionality.
    *   Form validation logic.

## 🌟 Features

*   **Responsive Design**: Fully adaptable layout that works on mobile, tablet, and desktop devices.
*   **Modern UI/UX**: Dark theme aesthetic with purple accents (`#8b5cf6`), featuring glassmorphism effects and smooth scrolling.
*   **Interactive Elements**: Hover effects on cards, floating animations, and dynamic navigation.
*   **Project Showcase**: A grid layout displaying my latest projects (e.g., Anime Explorer) with links to code and live demos.
*   **Contact Section**: Integrated form layout and social media links.
